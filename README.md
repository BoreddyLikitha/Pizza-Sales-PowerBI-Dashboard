# Pizza Sales Power BI Dashboard

## About the Project

This project is a Power BI dashboard created to analyze pizza sales data and understand business performance.  
The main goal was to practice data analysis, visualization, and dashboard design using real-world sales data.

## What This Dashboard Shows

- Total Revenue and Total Orders
- Monthly Sales Trend
- Best Selling Pizza Categories
- Sales by Pizza Size
- Order Quantity Analysis
- Top and Bottom Performing Products

## Tools Used

- Power BI Desktop
- DAX (for calculated measures)
- Data Modeling
- Basic Data Cleaning

## What I Learned

While working on this project, I improved my understanding of:
- Creating interactive dashboards
- Writing DAX measures
- Using slicers and filters
- Designing clean and readable reports
- Analyzing sales performance metrics
